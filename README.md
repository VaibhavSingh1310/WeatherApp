# 🌦️ Weather App

A simple and responsive weather application built using **HTML**, **CSS**, and **JavaScript**. It fetches real-time weather information using the **OpenWeather API**.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- 💧 Shows humidity
- 💨 Displays wind speed
- 📱 Responsive UI
- ⚡ Real-time weather data using Fetch API

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Navigate to the project folder

```bash
cd weather-app
```

3. Create a `config.js` file in the project root.

```javascript
const myApi = "YOUR_OPENWEATHER_API_KEY";
```

4. Open `index.html` in your browser.
